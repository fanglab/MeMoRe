MeMoRe documentation
====================

``MeMoRe`` stand for Methylation Motif Refiner. This is a tool designed to validate and, if necessary, refine methylation motifs detected from Bacteria and Archaea with SMRT and ONT sequencing.

Authors' notes
--------------

We are actively developing ``MeMoRe`` to facilitate usage and broaden features. All feedback is more than welcome. You can reach us on twitter (`@iamfanggang <https://twitter.com/iamfanggang>`_ and `@AlanTourancheau <https://twitter.com/AlanTourancheau>`_) or directly through the `GitHub issues system <https://github.com/fanglab/MeMoRe/issues>`_.

Usage
-----

To run ``MeMoRe`` on your own dataset or use our testing dataset please follow: https://fanglab-tools.shinyapps.io/MeMoRe/. ``MeMoRe`` is designed to be user friendly and therefore is distributed as a `Shiny <https://shiny.rstudio.com/>`_ application through `shinyapps.io <https://www.shinyapps.io/>`_ hosting solution, no installation needed.
